# data_warehouse_project
A data warehouse built with SQL server, involving ETL processes, data modelling/analysis and creating a database ready for ML use cases. 


 This project demonstrates a comprehensive data warehousing and analytics solution, from building a data warehouse to developing actionable insights. The project trys to 
 highlight best industry practices in data engineering.


 ## Project requirments

 ### Build a data warehouse 

 #### Objective 
 Develop a modern data warehouse using SQL server to consolidate sales data, which in turn can enable analytical reporting and decision making.

 #### Specifications 
 - **Data Sources**: Import data from two source systems (ERP and CRM) provided as csv files.
 - **Data Quality**: clean and reslove any data issues before analysis.
 - **Integration**: combine both sources into a single, user-friendly data model designed for analytical queries.
 - **Scope**: Focus on latest dataset; no historization of data needed.
 - **Documentation**: clear documentation of data model useful for business insights.

---

### BI: Analytics and Reporting (Data Analytics) 

#### Objective
Develop SQL analytics to provide insights into: 

- **Customer behaviour**
- **Product performance**
- **Sales Trends**
